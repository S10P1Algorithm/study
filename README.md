# 알고리즘 스터디
- 목표 : 플레?!
- 언어 : Python
- 기간 : 설 이전 까지

## 진행 방식
- 매주 백준에서 5 문제(멤버 각 1문제) 선정
- 화요일 3개, 목요일 2개 리뷰
- 문제 풀지 않을 시 1 문제당 1000원 벌금

## 공유 방식
- Github
- 폴더 구조 예시 : Week 1 - 7069 - 7069_홍길동.py
- 파일명: {{문제번호}}_{{이름}}

## 참고 사항
- 코드 작성시 변수명 및 함수명 등 이해할 수 있도록 설정
- 주석 상세히 작성
